**ถ้าเจนภาพจาก Auto journey และดาวน์โหลดภาพเก็บไว้ในโฟลเดอร์เรียบร้อยแล้ว จะต้องทำตามขั้นตอนนี้ก่อน**
- ทำการ Upscale ภาพในโฟลเดอร์ให้เรียบร้อย แนะนำให้ใช้โปรแกรม Lightroom
- ให้ทำการ Rename ไฟล์ภาพให้ถูกต้อง โดยการก็อบโปรแกรม ImageRename.exe และไฟล์ prompt.txt ไปไว้ในโฟลเดอร์เก็บภาพ
- เปิดไฟล์ prompt.txt และกรอกชื่อ username จาก discord ตรงบรรทัด user= เช่น user=user123
- กรอก prompt ที่ใช้เจนภาพทั้งหมดไว้ใต้บรรทัด //===================== เช่น
 //=====================
 prompt 1
 prompt 2
 prompt 3

- จากนั้น save file และปิดไป
- เปิดโปรแกรม ImageRename.exe ระบบจะทำการ Rename ให้รอจนกว่าจะเสร็จ แล้วปิดโปรแกรม

**ถ้าเจนภาพจากโปรแกรม auto-image.exe จนได้ไฟล์ภาพที่มีชื่อไฟล์ถูกต้องแล้ว หรือไฟล์ที่ผ่านการ Rename จากขั้นตอนก่อนหน้านี้ จะสามารถนำไฟล์ภาพมา Modify ใส่คีย์เวิร์ดต่อได้เลย
- ถ้าเจนภาพจากโปรแกรม auto-image.exe ให้ทำการ Upscale ภาพในโฟลเดอร์ให้เรียบร้อยก่อน
- ให้ก็อบโปรแกรม ImageModify.exe ไปไว้ในโฟลเดอร์เก็บภาพ 
- จากนั้นเปิดโปรแกรม และทำการ login เมื่อ login สำเร็จ ระบบจะทำการ Modify ใส่คีย์เวิร์ดให้ทุกรูป ให้รอจนกว่าจะเสร็จ แล้วปิดโปรแกรม

**เปิดโปรแกรม IMS เพื่อเช็คคีย์เวิร์ดก่อนส่งงานทุกครั้ง
